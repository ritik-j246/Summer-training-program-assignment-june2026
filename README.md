Hey! Welcome to my repository.

I am currently doing my college's summer training program during vacation. We are required to upload our daily coding assignments here to track progress and submit our coursework.

The main goal of this training is to learn the C programming language from the absolute ground up (zero prior knowledge required).

What I'm Learning:
Basics & Logic: Variables, data types, if-else conditions, and loops (for, while).

Core C Concepts: Functions, arrays, strings, and pointers.

Advanced: Structures, unions, and file handling.

How it’s Organized:
Everything is split into day-by-day folders (like Day-01, Day-02, etc.). Each folder contains the specific programs assigned to us by our instructors on that day.

Language: C

Tools: VS Code / GCC Compiler
